# Employee Management System (Spring Boot + JWT)

## 📌 Project Overview
This is a full-stack Employee Management System built using:
- Spring Boot (Backend)
- MySQL (Database)
- HTML, CSS, JavaScript (Frontend)
- JWT Authentication (Security)

---

## 🚀 Features

### 🔐 Authentication
- User Login (JWT Token based)
- Role-based access (ADMIN / USER)
- Password encryption (BCrypt)

### 👨‍💼 Employee Module
- Add Employee (Admin only)
- Update Employee (Admin only)
- Delete Employee (Admin only)
- View Employees (Admin + User)
- Search Employee by Name or Email

### 🏢 Department Module
- Add Department
- View Departments

### 🕒 Attendance Module
- Employee Check-in
- Employee Check-out
- Attendance History

---

## 🛠️ Tech Stack
- Java 17+
- Spring Boot
- Spring Security + JWT
- Spring Data JPA
- MySQL
- HTML/CSS/JS
